# Crok_seamless_GL

Creates seamless textures.


### INPUTS
* Source : Connect the image you want to apply the effect onto.

### HOW TO USE IT

#### Setup

* Version : Type of seamless effect..
  - Seamless 1
  - Seamless 2

#### Seamless 1

* Noise Scale : Noise scale.

#### Seamless 2

* Smoothness : Smoothness of the blending.
